# FilmTools — Competitive Comparison

> Prepared for CTO review.

Every professional tool that does what FilmTools aims to do is closed, expensive, or both:

| What we're doing | Closed equivalent | Cost |
|---|---|---|
| Spectral film simulation (agx-emulsion) | Dehancer, FilmConvert | $99–$149/yr |
| BRAW + R3D decoding (gpu-video) | Blackmagic/RED proprietary SDKs | License agreements |
| Gyro-assisted stabilization (Gyroflow Core) | DJI Mimo, Catalyst Browse | Proprietary |
| ACES 2.0 pipeline (OCIO 2.5 + Vulkan) | DaVinci Resolve Studio | $300 |
| SAM2 roto (open, offline) | Adobe Roto Brush, DaVinci Magic Mask | Subscription |
| **All of the above in one tool** | **Nothing** | **Doesn't exist** |

## The Opportunity

No single tool — open or commercial — combines all of these capabilities in one integrated pipeline. FilmTools is the first.
